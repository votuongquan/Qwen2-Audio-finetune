T có để 1 hình png trong repo á, để m coi cấu trúc input khi m chọn input trên kaggle sẽ giống vậy là ổn

Xong m clone repo của t về, muồn chỉnh gì đó thì tạo nhánh mới dưới local chỉnh sửa, sắp push lên, m chuyển branch như trên local rồi sài thôi. 
Khi m reload lại chỗ working dir sẽ thấy.

***Note***: Nhớ là mỗi lần chạy đều phải !pip install -r requirements.txt lại lần nữa nhá và cũng chỉnh config trên nhánh của m sao cho phù hợp và nhớ bật GPU

M muốn chạy file bash trên kaggle thì nó giống như !pip install trong code cell á:
!chmod +x <path_to_sh_file> (Vi du: !chmod +x train.sh) 

Xong thì chạy file thôi: !<path_to_sh_file> (Vi du:!./your_script.sh) 

Nay mai cứ coi code kĩ, hiểu rõ path các thứ là được. 

Trong setup utils, có file generate_scp.py để đẻ ra mấy cái file scp và generate_multitask.jsonl cho mấy cái multitask.jsonl á, m rảnh thì coi qua, để sau này cần sửa gì cũng tự sửa đc
